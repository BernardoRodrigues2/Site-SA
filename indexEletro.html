Introdução:


A eletroeletrônica estuda os componentes e sistemas usados para controlar e automatizar circuitos, desde elementos básicos como resistores e LEDs até placas programáveis como o Arduino. Ela envolve o uso de sensores, sinais digitais e analógicos, além de conceitos fundamentais como Lei de Ohm e potência elétrica. Esses conhecimentos são a base para criar projetos eletrônicos, sistemas automatizados e dispositivos inteligentes do dia a dia.


Componentes Eletrônicos empregados em serviços:


Resistor:
-> Diminui a passagem de corrente no circuito.
-> Usado para proteger componentes, ajustar tensão, limitar corrente de LEDs, etc.
Diodo LED:
-> Diodo que emite luz quando a corrente passa.
-> Tem polaridade (lado certo pra ligar).
-> Usado em indicadores, displays, iluminação simples.
Transistor:
-> Funciona como interruptor eletrônico ou amplificador.
-> Essencial para controlar cargas maiores, acionar motores, ler sinais, etc.
Circuito Integrado (CI):
-> Vários componentes dentro de um único chip.
-> Ex.: amplificadores operacionais, temporizadores (555), microcontroladores.

Sensores Empregados em Sistemas Automatizados:


São dispositivos que “sentem” alguma coisa no ambiente. Exemplos:
-> Temperatura: LM35, DHT11, termistor.
-> Umidade: DHT11, DHT22.
-> Luminosidade: LDR, fototransistor.
-> Proximidade: ultrassônico HC-SR04, infravermelho.
-> Gás: MQ-2, MQ-7.
-> Movimento: sensor PIR.

Arduíno: O que é, Para que serve / aplicações, vantagens e desvantagens.


O que é:
-> Placa eletrônica com microcontrolador, usada pra criar projetos eletrônicos e automação.
Para que serve:
-> Controlar sensores, motores, LEDs, relés, displays…
-> Usado em robótica, automação residencial, projetos escolares, IoT básico.
Vantagens:
-> Fácil de usar.
-> Barato.
-> Grande comunidade e exemplos prontos.
-> Compatível com muitos sensores.
Desvantagens:
-> Processamento limitado.
-> Não é ideal pra projetos muito complexos ou industriais.
-> Menos poderoso que Raspberry Pi, ESP32 etc.

Como o Arduíno funciona? O que é eletrônica embarcada?


-> O Arduino recebe sinais de sensores (entradas), processa isso no microcontrolador e aciona atuadores (saídas).
-> Ele funciona executando programas carregados pelo computador.
Eletrônica embarcada:
-> É quando sistemas eletrônicos + software estão “dentro” de dispositivos do dia a dia.
-> Ex.: carros, micro-ondas, impressoras, drones.

Arduíno: qual a linguagem de programação usada


-> Principal: C/C++ (Arduino IDE).
-> Também pode usar: Python (com MicroPython, dependendo da placa) e blocos (Scratch, Tinkercad), mas o padrão oficial É C/C++.

O que são entradas e saídas digitais em circuitos integrados como o Arduíno


Digitais = só dois estados:
-> 0 (LOW, 0V)
-> 1 (HIGH, 5V ou 3.3V)
Entradas digitais:
-> Leem se um botão está apertado, se um sensor enviou sinal, etc.
Saídas digitais:
-> Ligar/desligar LED, relé, buzzer, etc.

O que são entradas e saídas analógicas em circuitos integrados como o Arduíno


Analógicas = valores variando de forma contínua.
No Arduino:
-> Entradas analógicas leem valores de 0 a 1023 (10 bits).
-> Representam tensões de 0 a 5V.
Ex.: ler temperatura, luz, som.
Saída analógica:
-> Na verdade é PWM (não é analógico real).
-> Controla brilho de LED, velocidade de motor, etc.

O que é um diodo? e Um Diodo LED? e um display 7 segmentos?


Diodo:
-> Componente que deixa a corrente passar em apenas um sentido.
-> Usado para proteção, retificação, fonte, etc.
Diodo LED:
-> Mesma ideia do diodo, só que emite luz.
-> Precisa de resistor para não queimar.
Display de 7 segmentos:
-> Conjunto de 7 LEDs formando números de 0 a 9.
-> Muito usado em relógios, cronômetros, painéis simples.
